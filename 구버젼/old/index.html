주요 설정 파일의 위치

항목 설정 파일 위치 오토셋 매니저 메뉴
웹 서버의 설정 파일 C:\AutoSet6\Server\conf\httpd.conf 도구 > 웹서버 기본 설정
가상호스트 설정 파일 C:\AutoSet6\Server\conf\extra\httpd-vhosts.conf 도구 > 웹서버 세부 설정 > 가상 호스트 설정
PHP 설정 파일 C:\AutoSet6\Server\conf\php.ini 도구 > PHP 기본 설정
MySQL 설정 파일 C:\AutoSet6\Server\MySQL\my.ini 도구 > MySQL 설정
Sendmail 설정 파일 C:\AutoSet6\Server\sendmail\sendmail.ini 도구 > 센드메일 설정
